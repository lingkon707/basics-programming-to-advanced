1. Arithmetic Operators
+ : Addition
- : Subtraction
* : Multiplication
/ : Division
% : Remainder (Modulus)
** : Exponentiation
++ : Increment
-- : Decrement


2. Assignment Operators
= : Assignment
+= : Addition assignment
-= : Subtraction assignment
*= : Multiplication assignment
/= : Division assignment
%= : Remainder assignment
**= : Exponentiation assignment
<<= : Left shift assignment
>>= : Right shift assignment
>>>= : Unsigned right shift assignment
&= : Bitwise AND assignment
^= : Bitwise XOR assignment
|= : Bitwise OR assignment


3. Comparison Operators
== : Equal to
!= : Not equal to
=== : Strict equal to
!== : Strict not equal to
> : Greater than
< : Less than
>= : Greater than or equal to
<= : Less than or equal to


4. Logical Operators
&& : Logical AND
|| : Logical OR
! : Logical NOT


5. Bitwise Operators
& : AND
| : OR
^ : XOR
~ : NOT
<< : Left shift
>> : Right shift
>>> : Unsigned right shift


6. String Operators
+ : Concatenation
+= : Concatenation assignment


7. Conditional (Ternary) Operator
? : : Conditional (ternary) operator


8. Comma Operator
, : Comma


9. Unary Operators
typeof : Type of
void : Evaluates an expression without returning a value
delete : Deletes an object property
+ : Converts to a number
- : Negates the value


10. Relational Operators
in : Checks if a property is in an object
instanceof : Checks if an object is an instance of a class


11. Spread and Rest Operators
... : Spread syntax and Rest parameter


12. Optional Chaining Operator
?. : Optional chaining
